# Nami 项目背景
Nami这个app的目的：A cycle-sync lifestyle planner for women with PCOS and irregular cycles.

市场定位 - 周期 × 生活方式 × PCOS × 情绪能量管理
A cycle-sync lifestyle planner for women with PCOS and irregular cycles.
一个帮助 PCOS 和周期不规律女性，按生理周期安排生活的健康管理工具。

用户App上的操作路径：登陆 → 记录 → 今日建议 → 提醒 → 趋势洞察

1. Cycle-synced daily plan
 按周期阶段给生活建议：工作、运动、饮食、睡眠、情绪。 
2. PCOS-aware lifestyle intervention
 根据用户是否有胰岛素抵抗、瘦多囊、痘痘、经期不规律、压力敏感，给不同提醒。 
3. Fast logging + pattern insight
 每天 60 秒记录，7–14 天后给用户看到：睡眠/压力/饮食/运动和症状之间的关系。

Nami 的目标：
1. 高主动干预 × 高生活温度。目前没有竞品同时做到"真正懂你的生活节奏"和"主动告诉你今天该做什么"。
2. 产品核心是"荷尔蒙节律不是疾病，是你的潮汐，你要学会和它同行"。其核心理念是 “和月经周期同步安排生活” 。
